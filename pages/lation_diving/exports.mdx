### OpenMenu
Opens the main scuba diving menu.

#### Syntax

**Client**
```lua copy
exports.lation_diving:OpenMenu()
```

#### Examples  
```lua filename="client.lua" copy
exports.lation_diving:OpenMenu()
print('You opened the scuba diving menu!')
```
