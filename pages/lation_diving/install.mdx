## Install

Follow the steps below to install this resource to your server.
 
<Steps>
### Install dependencies
 
<Callout type="warning" emoji="⚠️">
    Verify all dependencies below are started *before* this script in your ```server.cfg```.
</Callout>

Install (or update) all dependencies below:

<Cards>
  <Cards.Card
    icon={<OpenInNew/>}
    title="ox_lib"
    href="https://github.com/overextended/ox_lib/releases/latest"
    target="_blank"
    rel="noopener noreferrer"
  />
</Cards>
 
### Add items
 
Open the ```install``` folder and add the ```items``` & their ```images``` to your inventory

<Callout type="info" emoji="ℹ️">
    You do not need to import the ```.sql``` file manually; it will be auto-imported.
</Callout>

<Tabs items={['ox_inventory', 'qb-inventory', 'qs-inventory']}>
  <Tabs.Tab>
    * Copy items from ```/install/items/ox_inventory.txt``` into ```ox_inventory/data/items.lua```
    * Copy images from ```/install/images``` into ```ox_inventory/web/images```
    ```lua
    -- Item data format for ox_inventory
    ['scuba_gear'] = {
        label = 'Scuba Gear',
        weight = 4750,
        stack = false,
        close = true
    },
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    * Copy items from ```/install/items/qb-inventory.txt``` into ```qb-core/shared/items.lua```
    * Copy images from ```/install/images``` into ```qb-inventory/html/images```
    ```lua
    -- Item data format for qb-inventory
    scuba_gear = { name = 'scuba_gear', label = 'Scuba Gear', weight = 4750, type = 'item', image = 'scuba_gear.png', unique = false, useable = true, shouldClose = true, combinable = nil, description = 'This looks useful' },
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    * **QBCore**: copy items from ```/install/items/qs-inventory.txt``` into ```qb-core/shared/items.lua```
    * **ESX**: copy items from ```/install/items/qs-inventory.txt``` into ```qs-inventory/shared/items.lua```
    * **Both**: copy images from ```/install/images``` into ```qs-inventory/html/images```
    ```lua
    -- Item data format for qs-inventory
    ['scuba_gear'] = {
        ['name'] = 'scuba_gear',
        ['label'] = 'Scuba Gear',
        ['weight'] = 4750,
        ['type'] = 'item',
        ['image'] = 'scuba_gear.png',
        ['unique'] = false,
        ['useable'] = true,
        ['shouldClose'] = true,
        ['combinable'] = nil,
        ['description'] = 'This looks useful'
    },
    ```
  </Tabs.Tab>
</Tabs>

Complete item list can be found in ```/install/items/*.txt```

### Restart server

* Add this script into your main resources directory (usually ```../resources```)
* Open ```server.cfg``` and add ```start``` or ```ensure lation_diving```
* Restart your server

</Steps>
