## Quasar Inventory

<Callout type="warning" emoji="⚠️">
    This guide is only necessary if you have metadata **enabled**!
</Callout>

<Steps>

### Step 1 - Add items

In ```qs-inventory/config/metadata.js```, go to the very top of the file and paste in the following code:

```js copy filename="metadata.js"
let lsWeedItems = [
    "ls_plain_jane_bud", "ls_plain_jane_joint", "ls_plain_jane_bag",
    "ls_banana_kush_bud", "ls_banana_kush_joint", "ls_banana_kush_bag",
    "ls_blue_dream_bud", "ls_blue_dream_joint", "ls_blue_dream_bag",
    "ls_purple_haze_bud", "ls_purple_haze_joint", "ls_purple_haze_bag",
    "ls_orange_crush_bud", "ls_orange_crush_joint", "ls_orange_crush_bag",
    "ls_cosmic_kush_bud", "ls_cosmic_kush_joint", "ls_cosmic_kush_bag"
];
```

<Callout type="error" emoji="⛔">
    If you rename your Weed Growing items (*or add new strains*) you **must** update this list ⬆️!
</Callout>
 
### Step 2 - Add metadata
Scroll down to line 46 (*or anywhere a code block ends like the picture below*) and place your cursor at the end
of the code block, it should look like this:  

![](/qs-metadata.png)  

Press enter to create a new line and paste the following code:
```js copy filename="metadata.js"
} else if (lsWeedItems.includes(itemData.name)) {
    $(".item-info-title").html("<p>" + `${itemData.info.label || label}` + "</p>");
    $(".item-info-description").html(
        "<p><strong>Purity: </strong><span>" +
        itemData.info.purity +
        "%</span></p>"
    );
} else if (itemData.name == "ls_watering_can" || itemData.name == "ls_fertilizer") {
    $(".item-info-title").html("<p>" + `${itemData.info.label || label}` + "</p>");
    $(".item-info-description").html(
        "<p><strong>Remaining: </strong><span>" +
        itemData.info.quality +
        "%</span></p>"
    );
```

### Step 3 - Save & restart
Done!
