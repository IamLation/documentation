## CodeM Inventory

<Callout type="warning" emoji="⚠️">
    This guide is only necessary if you have metadata **enabled**!
</Callout>

<Steps>

### Add items

In ```codem-inventory/config/metadata.js``` go to the very top of the file and paste in the following code:

```js copy filename="codem-inventory/config/metadata.js"
let lsWeedItems = [
    "ls_plain_jane_bud", "ls_plain_jane_joint", "ls_plain_jane_bag",
    "ls_banana_kush_bud", "ls_banana_kush_joint", "ls_banana_kush_bag",
    "ls_blue_dream_bud", "ls_blue_dream_joint", "ls_blue_dream_bag",
    "ls_purple_haze_bud", "ls_purple_haze_joint", "ls_purple_haze_bag",
    "ls_orange_crush_bud", "ls_orange_crush_joint", "ls_orange_crush_bag",
    "ls_cosmic_kush_bud", "ls_cosmic_kush_joint", "ls_cosmic_kush_bag"
];
```

<Callout type="error" emoji="⛔">
    If you rename your Weed Growing items (*or add new strains*) you **must** update this list ⬆️!
</Callout>
 
### Add metadata
In the same file search for ```driver_license```, you should see:
```js copy filename="codem-inventory/config/metadata.js" {8}
} else if (item.name.match("driver_license")) {
    let infoData = [
        { label: "Firstname", value: iteminfo.firstname || "Unknown" },
        { label: "Lastname", value: iteminfo.lastname || "Unknown" },
        { label: "Birthdate", value: iteminfo.birthdate || "Unknown" },
        { label: "Licenses", value: iteminfo.type || "Unknown" },
    ];
    returnString = infoData;
```

Place your cursor on the highlighted line above, hit enter to create a new line and paste the following code:
```js copy filename="codem-inventory/config/metadata.js"
} else if (lsWeedItems.includes(item.name)) {
    let infoData = [
        { label: "Purity", value: iteminfo.purity ? iteminfo.purity + "%" : "Unknown" }
    ];
    returnString = infoData;
} else if (item.name.match("ls_watering_can") || item.name.match("ls_fertilizer")) {
    let infoData = [
        { label: "Remaining", value: iteminfo.quality || "Empty" }
    ];
    returnString = infoData;
```

### Restart server
Done!
