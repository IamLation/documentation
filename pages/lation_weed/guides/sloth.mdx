### Add items

In ```ps-inventory/html/js/app.js``` search for the ```FormatItemInfo``` function, create a new empty line and paste in the following code:

```js copy filename="ps-inventory/html/js/app.js"
let lsWeedItems = [
    "ls_plain_jane_bud", "ls_plain_jane_joint", "ls_plain_jane_bag",
    "ls_banana_kush_bud", "ls_banana_kush_joint", "ls_banana_kush_bag",
```

<Callout type="error" emoji="⛔">
    If you rename your Weed Growing items (*or add new strains*) you **must** update this list ⬆️!
</Callout>

Once done, your ```FormatItemInfo``` function should look like this (*with the new addition highlighted*):

```js filename="ps-inventory/html/js/app.js" {3-10}

    let element = $('.ply-iteminfo-container');

        leftOffset = $(window).width() - element.width() - 20;
    }
```
