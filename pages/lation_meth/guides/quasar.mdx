## Quasar Inventory

<Callout type="warning" emoji="⚠️">
    You must be on ```qs-inventory``` version 2.7.8 or above!
</Callout>

<Steps>

### Find items

In ```qs-inventory/config/metadata.js``` search for ```ls_liquid_meth```, you should see:

```js copy filename="qs-inventory/config/metadata.js"
} else if (itemData.name == "ls_liquid_meth" || itemData.name == "ls_meth") {
```

<Callout type="error" emoji="⛔">
    If you rename your Meth Crafting items you **must** update them here as well!
</Callout>
 
### Add items
Highlight and replace the entire line found above with this:
```js copy filename="qs-inventory/config/metadata.js"
} else if (itemData.name == "ls_liquid_meth" || itemData.name == "ls_meth" || itemData.name == "ls_meth_tray" || itemData.name == "ls_meth_box") {
```

### Restart server
Done!
